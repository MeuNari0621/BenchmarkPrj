# 要求仕様書: アダプティブクルーズコントロール (ACC)

## 1. 概要

本文書は、車載ECU上で動作するアダプティブクルーズコントロール（ACC）の制御ロジックに対する要求仕様を定義する。
本ソフトウェアは、運転者が設定した目標車速を維持しつつ、先行車との車間距離を自動調整する機能を提供する。

## 2. システム境界

### 2.1 入力
- 自車速度センサ（0〜250 km/h、分解能 0.1 km/h）
- 前方レーダー（先行車との距離: 0〜200 m、先行車の相対速度: -50〜+50 m/s）
- ドライバー操作（ACC ON/OFF、設定車速変更、車間設定変更）
- ブレーキペダル信号
- アクセルペダル信号
- シフトポジション
- 車両故障診断情報

### 2.2 出力
- 加減速要求（-8.0 m/s² 〜 +2.0 m/s²）
- ACC動作状態
- 警報出力（視覚・聴覚）
- フォルトコード

### 2.3 除外事項
- アクチュエータ駆動制御（スロットル、ブレーキ）
- CAN通信スタック
- センサフュージョン
- 車線認識

## 3. 機能要求

### 3.1 状態遷移

#### FR-001: ACC状態管理
ACCは以下の状態を持つ:
- **STANDBY**: ACC無効、運転者による全操作
- **ACTIVE**: ACC有効、車速・車間制御中
- **OVERRIDE**: 運転者介入中（一時的にACC制御を中断）
- **FAULT**: 異常発生、ACC機能停止

#### FR-002: 状態遷移条件
- STANDBY → ACTIVE: ACC ONスイッチ押下 かつ 車速が30 km/h以上 かつ シフトがDレンジ かつ システム正常
- ACTIVE → STANDBY: ACC OFFスイッチ押下 または シフトがDレンジ以外
- ACTIVE → OVERRIDE: ブレーキペダル踏込 または アクセルペダルが設定車速相当の開度を超過
- OVERRIDE → ACTIVE: ブレーキ・アクセル解除後、2秒経過
- 任意状態 → FAULT: センサ異常 または 制御異常検出
- FAULT → STANDBY: フォルトリセット操作

### 3.2 車速制御

#### FR-003: 設定車速範囲
- 設定可能車速: 30〜180 km/h（1 km/h単位）
- デフォルト設定車速: 現在車速に最も近い5 km/h刻みの値

#### FR-004: 設定車速変更
- SET+スイッチ: 設定車速を+1 km/h（長押しで+5 km/h/秒）
- SET-スイッチ: 設定車速を-1 km/h（長押しで-5 km/h/秒）
- RESUMEスイッチ: 前回の設定車速に復帰

#### FR-005: 車速維持制御
- 先行車が検出されていない場合、設定車速を維持するよう加減速制御を行う
- 加速要求の上限: +2.0 m/s²
- 減速要求は車速維持制御では発生しない（先行車追従時のみ）

### 3.3 車間距離制御

#### FR-006: 車間時間設定
- 車間時間設定: 1.0秒、1.5秒、2.0秒、2.5秒（4段階）
- デフォルト: 1.5秒
- 目標車間距離 = 車間時間 × 自車速度

#### FR-007: 先行車追従制御
- 先行車が目標車間距離より近い場合、減速して車間を確保
- 先行車が目標車間距離より遠い場合、加速して設定車速まで回復
- 減速要求の上限: -3.5 m/s²（通常制動）

#### FR-008: 車間距離警告
- 実際の車間時間が設定車間時間の50%未満になった場合、視覚警告を出力
- 30%未満の場合、視覚警告+聴覚警告を出力

### 3.4 緊急制動

#### FR-009: 緊急制動介入条件
- 衝突予測時間（TTC）が1.5秒未満 かつ 先行車との相対速度が負（接近中）の場合、緊急制動を要求
- 緊急制動減速度: -8.0 m/s²まで許容

#### FR-010: 緊急制動解除条件
- TTC > 3.0秒 または 先行車がレーダー検出範囲外になった場合

### 3.5 運転者介入

#### FR-011: オーバーライド検出
- ブレーキペダル踏込（踏力センサ > 10 N）でオーバーライド
- アクセルペダル開度が「設定車速相当+5 km/h」を超える要求に相当する場合にオーバーライド

#### FR-012: オーバーライド復帰
- ブレーキ・アクセル両方が解除状態になってから2.0秒後にACTIVEへ復帰
- ただし、オーバーライド中に車速が25 km/h未満になった場合はSTANDBYへ遷移

### 3.6 フォルト処理

#### FR-013: センサフォルト検出
- 自車速度センサ: 値が範囲外（-1未満 または 251以上）または 500ms以上更新なし
- レーダー: 検出距離が範囲外 または 通信タイムアウト（200ms）

#### FR-014: フォルト発生時動作
- フォルト検出時、即座にFAULT状態へ遷移
- 加減速要求を0に設定（ドライバーへ制御を委譲）
- フォルトコードを記録

#### FR-015: フォルトリセット
- 全センサが正常値を1秒間継続 かつ イグニッションOFF→ON操作 でフォルトリセット可能

### 3.7 境界条件

#### FR-016: 勾配考慮
- 上り勾配では加速要求を最大+0.3 m/s²補正
- 下り勾配では減速要求を最大-0.3 m/s²補正
- 勾配は自車加速度と理論加速度の差から推定

#### FR-017: カーブ減速
- 横加速度が0.3Gを超えると予測される場合、設定車速の80%を上限に制限
- カーブの曲率はステアリング角と車速から推定

## 4. 非機能要求

#### NFR-001: 制御周期
- 制御周期: 20ms（50Hz）

#### NFR-002: 応答性
- ACC ON操作から制御開始まで: 100ms以内
- センサ異常検出からフォルト遷移まで: 40ms以内

#### NFR-003: 言語仕様
- C99準拠
- 動的メモリ割当禁止
- 浮動小数点演算使用可

#### NFR-004: テスト
- Google Testによる単体テスト
- 状態遷移カバレッジ100%
- 境界値テスト必須

## 5. 用語定義

| 用語 | 定義 |
|------|------|
| TTC (Time To Collision) | 衝突予測時間 = 車間距離 / |相対速度| |
| 車間時間 | 前車位置に到達するまでの時間 = 車間距離 / 自車速度 |
| オーバーライド | ドライバーがACC制御より優先して操作すること |

## 6. 補足

### 6.1 設計上の注意事項
本要求仕様には、実装者が注意すべき以下の観点が含まれている:
- 状態遷移の網羅性（全遷移パスのテスト）
- 複数条件の同時成立判定（AND条件の順序依存性）
- タイマー管理（2秒待機、500msタイムアウト等）
- 境界値の扱い（「以上」「未満」の区別）
- フォルト状態からの復帰条件の厳密性
- 単位系の一貫性（km/h, m/s, m/s²の混在）
